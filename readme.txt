Add a disclaimer to the post comment field of your site's comments.

== Description ==

Weaver Comment Disclaimer allows you to display a disclaimer or other terms right under the Post Comment button. Use
the "Weaver Comment Disclaimer" dashboard options menu to add whatever text you need. You can add basic HTML formatting,
and CSS style to the whole disclaimer since it is wrapped in a DIV tag.

== Installation ==

It is easiest to use the Plugin Add Plugin page, but you can do it manually, too:

1. Download the plugin archive and expand it
2. Upload the comment-disclaimer.php file to your wp-content/plugins/comment-disclaimer directory 
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' for Comment Disclaimer.

The control panel is found on the Dashboard Settings -> Weaver Comment Disclaimer menu.
